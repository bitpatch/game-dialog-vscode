# Changelog

## [0.1.0] - 2025-11-21

### Added
- Начальная версия расширения
- Подсветка синтаксиса для `.gds` файлов
- Поддержка комментариев (`//` и `/* */`)
- Подсветка операторов вывода диалогов (`<<`)
- Подсветка ключевых слов (`if`, `else`, `while`, `for`, и др.)
- Подсветка строк, чисел и операторов
- Автодополнение скобок и кавычек
- Сниппеты для быстрой вставки кода:
  - Вывод диалогов
  - Переменные
  - Условия
  - Циклы
- Иконка для `.gds` файлов
